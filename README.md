# FiveM-Updater
Update your FiveM server effortlessly.

## Prerequisites
### APT packages
**Debian/Ubuntu: Run <code>sudo apt update</code> then install with <code>sudo apt install \<package\></code>**
- python3
- python3-pip

### Python packages
**Install with <code>pip3 install \<package\></code>**
- requests
- beautifulsoup4
- gitpython
